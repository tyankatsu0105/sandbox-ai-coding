# セキュリティ

## 機密ファイル

読み取りや変更を行うな：

- .env ファイル
- \*_/config/secrets._
- \*_/_.pem
- API キー、トークン、または認証情報を含むすべてのファイル

## セキュリティ対策

- 機密ファイルをコミットするなではなく、コミットを禁止しろ。
- 秘密情報には環境変数を使用しろ。
- 認証情報はログや出力に含めるな。
- 機密情報を含むファイルは `.gitignore` に追加しろ。
