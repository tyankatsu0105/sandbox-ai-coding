## コマンド

プロジェクトで利用されるコマンドについて

### ローカル開発時

- `npm run dev`
  - 開発用サーバーを起動します。
- `npm run lint`
  - コードのリントを実行します。
- `npm run storybook`
  - Storybook を起動します。
- `npm run test`
  - テストを実行します。

### その他

- `npm run build`
  - プロダクション用のビルドを作成します。
- `npm run preview`
  - プロダクション用のビルドをプレビューします。
- `npm run build-storybook`
  - Storybook のビルドを作成します。
